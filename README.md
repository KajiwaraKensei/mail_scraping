# mail_scraping

メーリングリストをスクレイピングしてWebページに表示
## 使用言語・ライブラリ
 - Node@15.6.0
 - yarn@1.22.10
 - TypeScript
 - Next.js@11
 - styled-components
 - socket.io
 - puppeteer

## 使用方法

*.env.example* を参考に *./frontend* に *.env* ファイルを作成
```bash
cd frontend
yarn
yarn dev
```
[localhost:3000](localhost:3000)

## ビルド
```bash
yarn build
yarn start
```
